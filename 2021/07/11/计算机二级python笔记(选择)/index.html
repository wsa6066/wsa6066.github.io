计算机二级python笔记(选择)

发表于 2021-07-11 | 分类于 编程 ， python | 浏览 次

该篇文章主要是介绍计算机二级python的选择题部分，主要是记录一些知识点和易错点。

公共选择题

  • 在数据结构中，从逻辑上可以把数据结构分为线性结构和非线性结构

  • 前序、中序、后序遍历中的前中后指的是根节点遍历的顺序，前序就是根节点-左节点-右节点

    我们可以根据中序遍历和另一个遍历，确定二叉树和第三个顺序，如果给出了前序，则第一个就是根节点；后序同理，然后根据中序分割成两部分，继续上述操作。

  • 面向对象的开放方法中，类与对象的关系是抽象与具体

  • 软件没有明显的制作过程

  • 软件在运行期间不会因为介质的磨损而老化，只可能因为适应硬件环境以及需求变化进行修改而引入错误，导致失效率升高从而软件退化。

  • 笛卡尔积的定义是设关系R和S的元数分别是r和s，R和S的笛卡尔积是一个（r+s）元属性的集合。

  • 不是程序调试的基本步骤的选项是：分析错误原因

  • 补码的符号位取反即是偏移码

  • 计算机系统应该包括硬件系统和软件系统两大部分，其中硬件系统由运算器、存储器、控制器、输入设备和输出设备五大基本部件组成

  • 当CPU需要执行I/O操作时，程序将主动查询I/O设备是否准备好

  • 分时操作系统：允许多个联机用户同时使用一台计算机系统进行运算。（中央处理器的时间划分为时间片，轮流分给每个终端，每个用户在极短的时间内执行，若未执行完则需要等到下次分配的时间执行）

  • 操作系统的基本功能：处理机管理（进程管理）、存储管理、设备管理、文件管理、用户接口

  • 软件需求规格说明书不需要软件的可验证性，说明书内的可验证性是指每一个需求都在有限代价的有效过程中验证确认，对于软件没有可验证性。

  • 静态测试：不需要运行软件，通过人工进行分析
    动态测试：上机测试，通过运行软件来检测软件
    白盒测试：逻辑覆盖测试、基本路径测试
    所以基本路径测试属于：白盒测试方法且是动态测试

  • 概念数据模型：面向客观世界、面向用户的模型，与具体的数据库管理系统和具体的计算机平台无关

  • 数据库设计包括概念设计、逻辑设计和建立数据库。（不包括数据库管理系统）

  • 软件开发中需求分析的主要任务：定义和描述目标系统“做什么”

  • 在满足实体完整性约束的条件下：一个关系中应该有一个或多个候选关键词

  • 类对象的主要特征：对象唯一性

  • 高级语言根据计算机执行机制的不同可分成两类：静态语言和脚本语言，静态语言采用编译方式执行，脚本语言采用解释方式执行。（C语言是静态编译语言，python是脚本语言）

  • 二叉树中，度为0的节点数等于度为2的节点数加1

  • 软件的生命周期分为定义阶段、开发阶段、维护阶段。其中开发阶段包括设计、实现、测试，不包括可行性研究

  • [例题]设一棵树的度为3，其中度为3,2,1的结点个数分别为4,1,3。则该棵树种的叶子结点数为？

    设叶子结点的个数为n

    • 一方面总结点数为4+1+3+n
    • 一方面总结点数为3*4+2*1+1*3+0*n+1

    构建等式可得n=10

python选择题

  • 注意字符串str[i:j]其中包括第i个位置，不包括第j个位置

  • x={}生成的是空字典，不是空集合

  • 创建一个集合直接用{1，2, 3}，使用set(x)函数将其他的组合数据类型变成集合类型，返回结果是一个无重复且排序任意的集合。
    使用大括号可以创建字典，特殊地，可以创建一个空字典。
    d.items()返回字典中的所有键值对信息，返回结果是Python的一种内部数据类型dict_items。

  • python程序可以不包含main函数，main函数与其他函数地位相同

  • pip常用的子命令有：install、download、uninstall、freeze、list、show、search、wheel、hash、completion、help。

  • print(a,b)的时候注意输出的结果不是a,b，代码中的，只是分割的作用，不会输出

  • 字符串x大写应该使用x.upper()而不是upper(x)

  • [易错]print(str1.center(10,*))是错的，因为*没有被引用起来，正确的是print(str1.center(10,"*"))

  • python中复数加减运算，即使最后的结果是0也要写作0j

  • 存储多信息的数据,最适合的是列表数据类型,列表内含有多个元素,每个元素都是个单独的列表, 第一 个列表存储的是信息的分类, 每类作为个元素,后续列表存储的的都是同学的个人信息,将信息按照分类的顺序逐个存储即可。

  • 函数定义的时候并不执行，但是要放在调用它的语句之前定义

  • 文件的读写：r：只读模式，如果文件不存在，返回异常。x：创建写模式，文件不存在则创建，存在则返回异常。a：追加写模式，文件不存在则创建，存在则在原文件最后追加内容。+：与r/w/x/a一同使用，在原功能基础上增加同时读写功能。

  • 最适合字典类型表达的是：高维数据

  • 自己定义的函数名可以与Python内置函数同名，但容易出现错误，尽量避免。

常见函数

  • strip()移除字符串头尾指定的字符（中间部分不会删除）

    如果要删除str头尾的字符x：str.strip("x")

  • ord()和chr()函数是配对函数，chr将ASCII码转化为相应字符，ord将字符转化为对应的ASCII码

    eg:ord("a")=97 chr(97)=a

  • map()类似于映射，map(f,list)把f作用在list上

  • 列表的insert(方法必须要有两个参数,第一个参数是插入的位置 ,第二个参数是插入的元素。
