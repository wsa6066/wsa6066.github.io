Github+Hexo搭建个人网站

发表于 2021-03-08 | 分类于 网站相关 ， 搭建网站 | 浏览 次

第一篇文章主要介绍如何搭建这个网站，大致流程都是参考琛神的教程。Hexo是一款基于Node.is的静态博客框架，可以方便生成静态网页托管在Github上。

Github创建个人仓库

使用邮箱注册Github账号，点击Github的New repository创建仓库，仓库名为XXX.github.io，注意这个XXX即为Github的用户名，且最后生成的网址为https://XXX.github.io/。

安装Git

网站在本地搭建好了，需要使用Git同步到Github上，根据电脑的版本下载相应版本。安装完成后，为了不必每次发布网站时都要重新输入密码，可以将Git与Github账号绑定，鼠标右键打开Git Bash

title任意填即可，将pub文件内容复制粘贴到key内，即可将Git和Github绑定，在Git Bash中输入ssh git@github.com检测是否绑定成功。

安装Node.js

Hexo基于Node.js，在官网下载相应版本，安装后可在命令行输入node-v检测是否安装成功，输入npm-v检测npm是否安装成功。
