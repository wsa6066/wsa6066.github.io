SQL (Structured Query Language:结构化查询语言)，其主要功能是访问和处理数据库，包括数据的插入、查询、更新、删除。

首先介绍一下学习的资源及相关网站

  • SQL教程

  • 练习网站SQLZOO

开始之前先介绍一下注意事项：

  • 首先就是SQL对大小写不敏感，大写字母和小写字母是一样的。

  • SQL一般采用单引号环绕文本值（大部分也可以用双引号）
